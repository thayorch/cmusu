# Chiang Mai Student Union Website — Vue + Vite + Tailwind

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Stack

- **Vite 5** — build tool
- **Vue 3** — Composition API with `<script setup>`
- **Tailwind CSS 3** — utility-first styling
- **Google Fonts** — Kanit + Space Mono
